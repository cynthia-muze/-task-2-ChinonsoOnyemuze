DecodeLabs Data Analytics Internship - Project 2
Exploratory Data Analysis (EDA) on E-Commerce Sales Dataset

Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned e-commerce sales dataset provided as part of the DecodeLabs Data Analytics Internship Program.
The objective was to analyze customer purchasing behaviour, sales performance, payment preferences, referral effectiveness, and revenue trends while identifying patterns, outliers, and actionable business insights.

Dataset Information

1. Dataset Type: E-Commerce Sales Data
2. Total Records: 1,200 Orders
3. Total Columns: 14
4. Data Status: Cleaned and validated prior to analysis

Key Fields

1. OrderID
2. Date
3. CustomerID
4. Product
5. Quantity
6. UnitPrice
7. TotalPrice
8. PaymentMethod
9. OrderStatus
10. ReferralSource
11. CouponCode
12. TrackingNumber
13. ShippingAddress
14. ItemsInCart

Tools Used: Microsoft Excel & Microsoft Power BI

Data Preparation

Before analysis, the dataset was reviewed to ensure data quality.

The following checks were performed:

1. Missing values identified and handled
2. Duplicate records checked and removed where necessary

4. Text inconsistencies corrected

Sales Performance

1. The dataset generated 1.26 million euros in total revenue from 1,200 orders.
2. The average order value was 1,050.
3. The median order value was 824.

Customer Behaviour

1. The average order value exceeded the median order value, indicating the presence of higher-value purchases that increased overall spending averages.
2. Revenue trends fluctuated across the 30-month analysis period (January 2023 – June 2025).

Product Performance

1. Chair and Printer generated nearly equal and highest revenue, with Chair marginally ahead.
2. Phone generated the lowest revenue contribution.

Payment Method Analysis 

1. Online was the preferred payment option among customers.
2. Gift card recorded the lowest usage.

Referral Analysis

1. Instagram generated the highest number of orders.
2. Referral generated the fewest orders.

Revenue Trend

1. Revenue trends indicated periods of stronger customer purchasing activity during the analysis timeframe.
2. The dataset spans 30 months from January 2023 to June 2025.
3. Peak sales performance was observed in June, 2024.

Outlier Analysis

1. A small number of transactions recorded significantly higher order values than the median order value.
2. These high-value purchases contributed disproportionately to total revenue and increased the overall average order value.
